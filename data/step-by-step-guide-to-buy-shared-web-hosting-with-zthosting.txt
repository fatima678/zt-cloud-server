What is shared web hosting?
Shared web hosting is a class of
web hosting
where multiple sites are hosted by one physical server. Many customers use assets on the paired server, which keeps costs down. Customers each get a part of the server in which they can host their website files. Shared server hosts can handle hundreds of users. The overall cost of maintaining a server is spread over many users.
